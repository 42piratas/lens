# Nord Paper

Paper-on-dark variant of Nord. Canvas + sidebar stay on Nord's Polar Night (`nord0`..`nord3`); cards flip to Snow Storm (`nord4`..`nord6`) with `nord0` as ink. Strictly inside Nord's canonical palette — Polar Night on the dark stage, Snow Storm as the paper layer.

Mode: `dark-paper`. The shared `app/themes/_paper-pattern.css` does the structural work; this theme only declares the 10 paper-layer tokens. See `app/themes/README.md` → "Paper-on-dark pattern".

Source: `https://www.nordtheme.com`
