# 42labs Dark Paper

Paper-on-dark variant of `dark` (42labs DS). Page canvas + sidebar stay dark (void / panel); tile bodies and inner cards flip to the dark palette's light-end metal tones (steel / fog) — strong contrast on cards without leaving the brand palette.

Mode: `dark-paper`. The shared `app/themes/_paper-pattern.css` does the structural work; this theme only declares the 10 paper-layer tokens. See `app/themes/README.md` → "Paper-on-dark pattern".

Source: `https://42labs.io/design`
