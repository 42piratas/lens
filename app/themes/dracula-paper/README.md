# Dracula Paper

Paper-on-dark variant of Dracula. Canvas + sidebar stay on Dracula's `Background` / `Current line`; cards flip to Dracula's `Foreground` (cream-white) with `Background` (`#282a36`) as ink. A lavender-tinted muted-paper tone (`#d8d6e0`) carries the multi-card tile body so the Dracula purple atmosphere persists on the paper side.

Mode: `dark-paper`. The shared `app/themes/_paper-pattern.css` does the structural work; this theme only declares the 10 paper-layer tokens. See `app/themes/README.md` → "Paper-on-dark pattern".

Source: `https://draculatheme.com`
