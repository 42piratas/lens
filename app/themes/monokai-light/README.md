# Monokai Light

Light-mode inversion of the canonical Monokai (Wimer Hazenberg). Warm off-white surface (#fafafa) with the same chromatic accents — pink, green, orange, cyan, purple — darkened just enough to meet AA contrast on a light background. Comment grey (#75715e) is preserved verbatim so muted text still reads as "Monokai."

Pairs with the dark `monokai` theme as the user's preferred light/dark switch.
