# Monokai Paper

Paper-on-dark variant of Monokai. Canvas + sidebar stay on canonical Monokai (`#272822` bg + `#1e1f1c` alt + `#3e3d32` selection); cards flip to Monokai Light's warm off-white (`#fafafa`) with canonical Monokai bg (`#272822`) as ink. The Monokai signature pink (`#f92672`) accent stays shared across both contexts.

Mode: `dark-paper`. The shared `app/themes/_paper-pattern.css` does the structural work; this theme only declares the 10 paper-layer tokens. See `app/themes/README.md` → "Paper-on-dark pattern".

Source: `https://monokai.pro`
