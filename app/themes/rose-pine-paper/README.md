# Rosé Pine Paper

Paper-on-dark variant of Rosé Pine. Canvas + sidebar stay on Rosé Pine **Main** (`base` / `surface` / `overlay`); cards flip to **Dawn**'s `base` (`#faf4ed` cream) with Dawn's `text` (`#575279` dusty purple) as ink. Strictly inside the Rosé Pine canonical palette — Main on the dark stage, Dawn as the paper layer.

Mode: `dark-paper`. The shared `app/themes/_paper-pattern.css` does the structural work; this theme only declares the 10 paper-layer tokens. See `app/themes/README.md` → "Paper-on-dark pattern".

Source: `https://rosepinetheme.com`
